## OS Project
<hr>

###### Use shell script for `generating the file list` in a shell function
<hr>

**Contents of files :**


 file.sh : 
 
 simple for loop used for generating file list
 
 ls.sh :
 
 ls command used for generating file list
 
 ls_while.sh : 
 
 ls and while used simultaneously
 
 advance :
 
 it has various types of flags for different viewing of files, like their sizes, names, types, etc.

 shell_function.sh :
 
 shell functions are used for two different methods of listing the files. Users are given the choice to move fwd with anyone option. And the    script also shows execution time and real time clock. This script has the interactivity.
 <hr>
 <br>
 
 **For running any script follow the following commands:**
 
 
  $```chmod +x script.sh ``` #change the name script to file name
  
  $``` ./script.sh```
